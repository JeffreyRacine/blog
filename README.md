# Jeffrey S. Racine's Personal Blog

This repository contains the quarto files for my blog (migration August 11 2023 while in Pretoria, SA). This is a migration from my 2018 blog created using R blogdown and netlify (hosting). In the current iteration I am using GitHub Pages for hosting which saves a step since I host the files on GitHub in either case). Blogdown and the hugo theme were not aging well and netlify (to its credit) was issuing security vulnerability warnings. I have terminated the netlify instance and eventually search bots will pick up this new one at its new location.
